# University_Library_Application
App quản lý thư viện môn học Android
# Chức năng người dùng
- Xem sách
- Tìm sách
- Lọc sách theo loại
- Tìm kiếm
- Mượn sách
- Trả sách
- Thêm list yêu thích
- Thống kê
# Chức năng thủ thư
- Duyệt mượn
- Duyệt trả
- Duyệt tài khoản
